What is DappsWallet?

DappsWallet is an open source protocol for connecting
decentralised applications to mobile wallets with QR code scanning
or deep linking. A user can interact securely with any Dapp from
their mobile phone, making DappsWallet wallets a safer choice
compared to desktop or browser extension wallets.

How does it work?

DappsWallet connects web applications to supported mobile
wallets DappsWalletsession is started by a scanning a QR code
(desktop) or by clicking an application deep link (mobile).
